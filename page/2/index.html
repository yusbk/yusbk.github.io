Merge Selected File

This guide is from StackOverflow on how to selectively merge files form one branch to another. Basically it’s: git merge --no-ff --no-commit dev Where dev is the branch you want to merge from into the current branch. The --no-commit option will stage the files that have been merged by Git without actually committing them.
