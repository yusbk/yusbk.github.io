Susun factor

Susun ketinggian bar Nama di aksis Bila buat bar plot cara asas di R atau menggunakan pakej ggplot2, susunan levels di x-aksis disusun semula menggikut abjad. Oleh sebab itu sekiranya ingin menyusun mengikut aturcara tertentu, levels di x-aksis harus disusun semula. Pelbagai cara untuk menyusun variable boleh dibuat contohnya menggunakan order (dari “base”) dan reorder (dari “stat”). Kedua-dua fungsi ini sudah tersedia di R tanpa perlu memuat turun sebarang pakej.
