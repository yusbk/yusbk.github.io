Smerge and ediff

For enditing conflict in git, use smerge or ediff. Smerge Basic workflow: Open magit status with SPC g g in Doom Identify the mark with Unmerged under Unstage list Go to the file and press Enter to open the file The base command for Smerge is C-c ^ You will see the conflict and selecting the upper with C-c ^ u and lower with C-c ^ l To select all ‘conflicting’ changes press C-c ^ a To navigate the conflicts use C-c ^ n and C-c ^ p for next and previous conflict To see all the keybindings use C-c ^ h Ediff When you are in the file with conflict, type M-x smerge-ediff or M-x vc-resolve-conflicts.
