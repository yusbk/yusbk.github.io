Plot dan table

Ini ialah aturaca membuat plot dan memasukkan table untuk informasi berkaitan plot menggunakan pakej ggplot2. Table untuk informasi juga akan dibuat menggunakan ggplot2 yang kemudiannya digabungkan menggunakan pakej gridExtra. Untuk memastikan plot dan table akan sejajar, adalah penting untuk menggunakan skala yang sama. Skala ini boleh menggunakan nombor atau kumpulan (category). Bila menggunakan skala secara category adalah penting untuk menukarkan status kolom ke factor. Contoh data: library(data.table) library(ggplot2) navn <- c("Nama1", "Nama2 yang panjang", "Nama3") N <- c(112, 210, 150) N2 <- c(200, 250, 170) dat <- data.
