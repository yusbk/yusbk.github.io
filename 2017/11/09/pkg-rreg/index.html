R pakej `rreg`

 Nov 9, 2017      R  my 

Pakej rreg sudah dimuatkan di CRAN pada 8 Ogos 2017. Pakej ini dibuat untuk memudahkan kerja-kerja visualisasi untuk data-data daripada pendaftaran kesihatan di Nasjonalt ServicemiljÃ¸ for medisinske kvalitetsregistre. Untuk memakai pakej rreg boleh dipasang secara langsung dari CRAN.

install.packages("rreg", dependencies = TRUE)

Pakaj ini menggandungi beberapa fungsi untuk visualisasi data. Fungsi-fungsi berikut ialah:

  1. Bar chart dengan regbar
  2. Radar diagram menggunakan regrad
  3. Untuk menunjukkan tren dengan regline
  4. Menggunakan “confidence interval” atau “standard error” dengan regerr

Contoh penggunaan boleh dilihat dengan melarikan contoh-contoh yang tersedia dengan menggunakan fungsi example.

library(rreg)
example(regbar)
plot
