R pakej `rreg`

Pakej rreg sudah dimuatkan di CRAN pada 8 Ogos 2017. Pakej ini dibuat untuk memudahkan kerja-kerja visualisasi untuk data-data daripada pendaftaran kesihatan di Nasjonalt Servicemiljø for medisinske kvalitetsregistre. Untuk memakai pakej rreg boleh dipasang secara langsung dari CRAN. install.packages("rreg", dependencies = TRUE) Pakaj ini menggandungi beberapa fungsi untuk visualisasi data.
