Pemakaian pekej Grid

Pakej grid dipakai untuk membuat grafik secara asas membolehkan pemakaian aturcara dan peraturan yang lebih anjal mengikut kemahuan sendiri. Ia juga boleh dipakai untuk mengatur kedudukan grafik. Fungsi penting yang dipakai ialah: viewport pushViewport grid.draw viewport dipakai untuk menetapkan kawasan grafik. Fungsi viewport() digunakan untuk tujuan ini. Contohnya: library(grid) vp1 <- viewport(x = 0.5, y = 0.5, width = 0.2, height = 0.4, just = c("left", "bottom"), name = "vp1") grid.
