Plot dan table

Ini ialah aturaca membuat plot dan memasukkan table untuk informasi berkaitan plot menggunakan pakej ggplot2. Table untuk informasi juga akan dibuat menggunakan ggplot2 yang kemudiannya digabungkan menggunakan pakej gridExtra. Untuk memastikan plot dan table akan sejajar, adalah penting untuk menggunakan skala yang sama.
